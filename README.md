# recipe-recommender
A Python-based recipe recommendation system that recommends meals based on available ingredients, dietary preferences, and calorie limits.

Many users struggle to decide what to cook based on the ingredients they already have at home.
This project addresses that problem by recommending recipes that match available ingredients and dietary needs.

Main goals:
- recommend recipes based on ingredients
- support dietary filtering
- consider calorie limits
- rank recipes by relevance

Features:
- ingredient-based recipe recommendation
- dietary preference filtering
- calorie-aware recipe selection
- missing ingredient detection
- ranked top-N recommendations
- optional Streamlit interface

Dataset:
The dataset was downloaded from Kaggle.

Tools used:
- Python
- pandas
- NumPy
- scikit-learn
- Streamlit
